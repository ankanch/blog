# index

【本应用非腾讯官方应用，与腾讯无任何关系】 本应用可以在手机上实现对腾讯云资源的管理。目前支持：

***云服务器管理**：开关机，重启，重装系统，重设密码

***私有镜像管理**：创建云服务器镜像，从镜像创建服务器

***创建实例：**创建/退还按量使用云服务器实例

***域名管理：**增加，修改，删除域名解析记录
